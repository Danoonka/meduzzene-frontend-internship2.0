## Internship knowledge control

### First step:
Fork repository by link https://github.com/Danoonka/meduzzene-frontend-internship2.0

### Second step:
Install dependency using command:

 ```
    npm install
```

### Third step: 
run commands

```
  cd my-app
  ng serve --open
```

App will start on http://localhost:4200/
